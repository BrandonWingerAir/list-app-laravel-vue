# List App (Laravel & Vue)

Note: Requires Node.js installation.

## Getting Started

Configure .env file with your database.

In the project directory, you can run:

### `npm install`

### `php artisan migrate`

### `php artisan serve`

### `npm run hot`

Runs the app in the development mode.\
